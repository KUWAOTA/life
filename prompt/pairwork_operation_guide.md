# Pairwork + Splitwise 運用ガイド

> **運用フロー図**: 同梱の draw.io 図を参照

---

## ツール構成

| ツール | 役割 | 費用 |
|--------|------|------|
| **Pairwork**（プレミアム） | 家事の記録・時給換算・貢献度可視化 | ¥480/月 |
| **Splitwise** | 実費（家賃・光熱費・食費等）の割り勘管理 | 無料 |

---

## 毎日やること

1. **家事を完了したら** → Pairworkで完了タップ（時給換算が自動で貯まる）
2. **実費が発生したら** → Splitwiseに記録

これだけ。

---

## 月末にやること（3ステップ）

**Step 1**: Pairworkの月次グラフで各自の家事額（時給換算）の差分を確認

**Step 2**: 差分額をSplitwiseに「家事精算 ○月分」として登録
```
例: 彼女が¥3,800多くやった → 彼氏→彼女 ¥3,800 を登録
```

**Step 3**: Splitwiseの精算画面で実費+家事差分の合計を確認 → 振込

## Pairwork 家事登録ルール

### 時給設定
**時給1,200円固定**。所要時間を調整してJen額に合わせる。

> 計算式: 所要時間（分） = Jen ÷ 1,200 × 60

### daily quest → 繰り返し設定あり
名前はそのまま登録。メモ欄にJen額と完了条件を記載。

### guerrilla quest → 繰り返し設定なし
名前の先頭に `[G]` を付ける。やった人がやった時に記録。

### 食事の特別ルール
- 平日夜の食事準備・片付け → daily
- 土日の食事準備・片付け → guerrilla（彼氏不在の日が多いため）
- 準備と片付けは同数割り振り

## PCからPairworkに家事を登録する方法

Pairwork には Web版がない（モバイルアプリのみ）。PCから操作するには以下の方法がある:

### 方法1: Phone Link（スマートフォン連携）— おすすめ

Windows 11 標準機能。スマホ画面をPCにミラーリングし、マウス・キーボードで操作できる。

- **対応機種**: Galaxy端末はフル対応（アプリ操作可）。Pixel等は通知・写真のみで画面操作は非対応の場合あり
- **設定**: Windows「スマートフォン連携」アプリ → スマホでQRコード読み取り → 接続
- **メリット**: エミュレータ不要。追加インストールなし。レイテンシも低い
- **デメリット**: Galaxy以外はアプリ操作に制限がある場合あり

### 方法2: Androidエミュレータ（BlueStacks）

PCにAndroid環境を丸ごと入れる方法。端末を問わず使える。

- **推奨**: [BlueStacks 5](https://www.bluestacks.com/) （GDPR準拠、最も軽量）
- **手順**: BlueStacksインストール → Google Playにログイン → Pairworkをインストール
- **メリット**: どのスマホでもOK。キーボード入力で家事登録が速い
- **デメリット**: 常駐でメモリを消費（2〜3GB）。Pairworkアカウントの同時ログイン可否は要確認

### 方法3: scrcpy（開発者向け）

Android端末をUSB/WiFiでPCにミラーリングするオープンソースツール。

- **メリット**: 軽量、どのAndroid端末でも動作、遅延が極めて少ない
- **デメリット**: USBデバッグの有効化が必要（開発者オプション）

### おすすめ判断

| 状況 | 推奨方法 |
|------|---------|
| Galaxyスマホを使っている | Phone Link |
| Galaxy以外 + 技術に詳しい | scrcpy |
| Galaxy以外 + 手軽に使いたい | BlueStacks |

## 自作アプリへの移行判断

2〜3ヶ月運用して以下に該当すれば自作を検討:

- [ ] 月次精算の手動作業が毎回ストレス
- [ ] 2アプリの行き来が面倒で記録が続かない
- [ ] Pairworkの時給換算とJen額のズレが気になる

回っているなら自作は不要。
