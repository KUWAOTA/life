# 類似アプリ調査レポート

## 調査目的

「家計の収支管理」と「家事タスク（Quest）の報酬（Jen）管理」を統合し、家事報酬が家計に反映されるアプリの構想に対して、既存の類似アプリ・サービスを調査した。

---

## 要件との適合度マトリクス

構想アプリの主要要件:
1. **家事タスク管理** — daily/guerrillaの区分、完了条件付き
2. **報酬（Jen）システム** — 家事に仮想通貨的な報酬を紐付け
3. **家計収支管理** — 実際の支出（Yen）の記録・割り勘
4. **家事報酬 → 家計反映** — 家事の報酬差分が家計精算に組み込まれる
5. **ゲーミフィケーション** — Quest呼称、モチベーション維持
6. **協議機能** — 分担の見直しをアプリ内で行える

| アプリ | 家事管理 | 報酬/ポイント | 家計管理 | 家計連動 | ゲーミフィケーション | 協議機能 |
|--------|:--------:|:------------:|:--------:|:--------:|:------------------:|:--------:|
| **Shareroo** | O | X | O | X | X | X |
| **Pairwork** | O | O (時給換算) | X | X | X | X |
| **CAJICO** | O | O | X | X | O | X |
| **Nipto** | O | O | X | X | O | X |
| **Splitwise** | X | X | O | X | X | X |
| **Walica** | X | X | O (割り勘) | X | X | X |
| **構想アプリ** | **O** | **O** | **O** | **O** | **O** | **O** |

> **結論: 「家事報酬を家計精算に統合する」アプリは現時点で存在しない。** これが構想アプリの最大の差別化ポイント。

## 類似アプリ詳細

### 1. Shareroo — 最も近い統合型アプリ

**適合度: ★★★★☆（家計+タスク管理の統合はあるが、報酬連動なし）**

- **概要**: 予算管理とタスク管理を1つのアプリで提供。100万人以上のカップル・家族が利用。
- **家計機能**: 支出追加、割り勘（均等/比率/金額指定）、ワンタップ精算、レシートOCR
- **タスク機能**: 共有To-Doリスト、期限通知、タスク割り当て
- **価格**: 無料（基本機能）
- **欠点**: 家事に報酬が紐づかない。タスク完了が家計に影響しない。ゲーミフィケーションなし。
- **参照**: [App Store](https://apps.apple.com/us/app/shared-budget-planner-shareroo/id1475406336)

---

### 2. Pairwork（ペアワーク） — 家事の金銭価値化で最も近い思想

**適合度: ★★★★☆（家事の時給換算は近いが、家計管理・精算機能なし）**

- **概要**: 株式会社mediba開発。共働き夫婦向け家事分担アプリ。特許取得済み（特許第7717209号）。
- **特徴的機能**:
  - **時給換算**: 家事の労働価値を時給に換算して可視化（構想アプリの「Jen」と類似思想）
  - **AI分析**: 家事分担バランスをAIが分析しコメント
  - **150以上のプリセット家事**: カスタム追加も可能
  - **貢献度の円グラフ表示**
  - **感謝スタンプ機能**
- **価格**: 無料
- **欠点**: 家計管理機能がない。報酬が家計に反映されない。ゲーミフィケーション要素が弱い。
- **参照**: [公式サイト](https://pairwork.app/) / [App Store](https://apps.apple.com/jp/app/%E3%83%9A%E3%82%A2%E3%83%AF%E3%83%BC%E3%82%AF-%E5%AE%B6%E4%BA%8B%E5%88%86%E6%8B%85-%E6%99%82%E7%B5%A6%E6%8F%9B%E7%AE%97%E3%82%A2%E3%83%97%E3%83%AA/id6464381260)

---

### 3. CAJICO（カジコ） — ポイント＆報酬システムで最も近い

**適合度: ★★★☆☆（ポイント→ご褒美交換はあるが、家計連動なし）**

- **概要**: 「ありがとうが見える」家事共有アプリ。家事・育児の見える化。
- **特徴的機能**:
  - **ポイント制**: 家事にポイントを設定、完了で獲得
  - **ご褒美交換**: 貯めたポイントを事前設定したご褒美と交換
  - **タスクスケジュール管理**: 定期タスクの繰り返し設定
  - **完了通知**: パートナーに家事完了を通知
  - **自由カスタマイズ**: タスク・ポイント・ご褒美を自由に設定
- **価格**: 無料
- **欠点**: ポイントは仮想的なもので日本円に換算されない。家計管理機能なし。daily/guerrillaのような分類なし。
- **参照**: [公式サイト](https://cajico.com/) / [Google Play](https://play.google.com/store/apps/details?id=com.herokuapp.cajico)

---

### 4. Nipto — ゲーミフィケーション特化

**適合度: ★★★☆☆（ポイント競争は近いが、家計機能なし）**

- **概要**: 家事をゲームに変えるアプリ。週ごとのポイント競争。
- **特徴的機能**:
  - **ポイント制**: 家事完了でポイント獲得
  - **週間ランキング**: 毎週日曜に勝者決定
  - **カスタム報酬**: 勝者が受け取る報酬をカップルで設定（映画選択権、買い物免除など）
  - **定期タスクグループ**: 繰り返しスケジュール設定
- **価格**: 無料 / $5.99/月（プレミアム）
- **欠点**: 報酬は金銭的ではなくカジュアルな「ご褒美」。家計管理機能なし。
- **参照**: [公式サイト](https://nipto.app/) / [App Store](https://apps.apple.com/us/app/nipto-split-chores/id1504877473)

---

### 5. Splitwise — 家計管理のスタンダード

**適合度: ★★☆☆☆（家計管理は優秀だが家事機能なし）**

- **概要**: 割り勘・共有支出管理の定番アプリ。
- **特徴的機能**:
  - 柔軟な割り勘方法
  - グループ管理、複数通貨対応
  - 定期請求、CSV出力
  - 精算の最適化
- **価格**: 無料（基本） / Pro版あり
- **欠点**: 家事管理機能は一切なし。
- **参照**: [公式サイト](https://www.splitwise.com/)

### 6. Walica（ワリカ） — プロンプトで言及された参考アプリ

**適合度: ★★☆☆☆（旅行割り勘特化）**

- **概要**: 旅行時の割り勘計算に特化したWebアプリ。
- **特徴**: 登録不要、シンプルな精算計算
- **欠点**: 日常の家計管理や家事分担には非対応。旅行向け。
- **参照**: [公式サイト](https://walica.jp/)

## Gap分析：構想アプリのユニークな価値

### 既存アプリでカバーされている領域
- 家事タスクの登録・管理・通知
- ポイント/報酬による家事のゲーミフィケーション
- 家事の金銭的価値への換算（Pairwork）
- カップル間の家計・割り勘管理

### 既存アプリでカバーされていない領域（＝構想アプリの独自価値）
1. **家事報酬（Jen）と家計収支（Yen）の統合精算** — 家事をやった分が実際の金銭精算に反映される仕組みは存在しない
2. **Daily Quest / Guerrilla Quest の区分** — 定期家事と不定期家事で報酬バランスのルールを変える設計は見られない
3. **差分ゼロ設計** — daily questに限り報酬差分が0になるよう分担を設計する思想は独自
4. **協議機能** — 分担の見直しをアプリ内で提案・合意するUIは既存アプリにない

## 運用で代替する場合の候補

自作せず既存アプリで運用する場合、以下の組み合わせが最も近い:

### 案A: Pairwork + Splitwise
- **Pairwork**: 家事の分担管理・時給換算で貢献度を可視化
- **Splitwise**: 実際の支出を記録・精算
- **運用ルール**: 月末にPairworkの差分をSplitwiseに手動で反映
- **課題**: 2アプリの運用が面倒。Jen→Yen変換が手動。

### 案B: CAJICO + Warikani
- **CAJICO**: 家事ポイント管理・ご褒美システム
- **Warikani**: ふたり専用の割り勘・家計簿
- **運用ルール**: CAJICOのポイント差分を月末にWarikaniで精算
- **課題**: やはり2アプリ管理。日本語UIで馴染みやすい。

### 案C: Shareroo 単体
- **メリット**: 1アプリで家計+タスク管理が完結
- **課題**: 家事報酬の概念がないため、ポイント制は別途スプレッドシートなどで管理が必要

---

## 結論

**「家事の報酬を家計精算に統合する」アプリは現時点で市場に存在しない。**

最も思想が近いのは **Pairwork**（家事を時給換算）と **CAJICO**（家事にポイント付与＋ご褒美交換）だが、いずれも実際の金銭精算との連動は行わない。

構想アプリが実現すれば、「家事の見える化」と「家計の公平な精算」を一気通貫で解決する初のアプリとなる可能性が高い。

ただし、**MVP段階では既存アプリ（Pairwork or CAJICO）+ 割り勘アプリ（Splitwise or Warikani）の組み合わせで運用を開始**し、不足点を洗い出してから自作に移行するアプローチも検討に値する。
