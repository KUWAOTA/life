# カップルプロファイル：拓磨 × 萌子

> このファイルは、二人のコミュニケーション特性・衝突パターンを記録し、
> 次回の振り返りに活かすためのリファレンスです。
>
> **重要な前提：** 二人とも論理的思考力を持ち、論理的な解決を好む。
> 以下に記述する差異は「論理 vs 感情」ではなく、**ストレス下で処理モードがどう変化するか**、
> **何を入力チャンネルとして優先するか**の違いである。

---

## 拓磨

### 基本スタイル
- **論理的分析に強い**。構造化・抽象化が得意で、問題を素早く分類できる
- フォーカスを言葉で明示的に設定する（「この話は〇〇の問題だ」）
- 「〜と思う」を付けて主観を明示化する習慣がある
- 感情を「原因から生じる一時的リアクション」と位置づけ、原因の特定と構造的解決を志向する

### ストレス下で起きること
拓磨も感情の影響を受けるが、それが**分析的行動の形で表出する**ため、本人には「論理的に対処している」と見える。実際には：
- **プロセス圧縮**：感情負荷が高い時ほど「早く解決したい」衝動から、分析を一人で完結させて結論だけを提示する傾向が強まる
- **枠組みの一方的宣言**：不安や焦りが駆動しているが、発言形式が論理的なので自覚しにくい
- **帳簿化**：「折れる」「覚えておいて」は譲歩への不満の表出だが、論理的コーティングがされている
- **母親記憶の連結**：「理不尽な命令」と感じた瞬間、過去の記憶と接続して怒りが増幅する
- 感情的洪水の相手に対しても分析モードを維持し続ける（→相手には圧力として受信される）

### 強み
- 問題の本質を素早く把握できる
- 自分から譲歩・折り合いを提案できる
- メタ視点から対話を俯瞰できる
- 怒りの衝動を観測対象に変換してログ化できる
- 「認知負荷と一時感情負荷を両方考慮する必要がある」という設計上の洞察を自ら導出している

## 萌子

### 基本スタイル
- **論理的思考力がある**。実験科学者（児童心理）として、仮説検証・エビデンスベースの思考を日常的に行う
- 本セッションの分析に全行程参加し、論理的に正確な修正を加えている（時給換算は既存制度、味の問題であり健康ではない、等）
- 「作者ルール」（利害マップ上の最適解）を提案したのは萌子である
- 「多数決に頼るのは、他に合理的に示す余地がないから」という自己分析はメタ認知的
- 非言語シグナル（表情・雰囲気）からの情報取得が得意。これは言語チャンネルの代替ではなく、追加の入力チャンネル

### ストレス下で起きること
萌子はストレス下で**論理的処理から関係安全性の確保へモードが切り替わる閾値が、拓磨より低い**。これは能力の差ではなく、閾値の差：
- **正当性の証明を求められた時**：論理の土俵で自分の主張に根拠がないと感じると、外部データ（統計・多数決）でエビデンスを調達しようとする。これは科学者としての論理的行動だが、二人の対話文脈では「合理的に示す余地がない」状態の防衛反応でもある
- **主張が繰り返し却下された時**：処理モードが「問題解決」から「関係の安全確認」に切り替わる
- **感情的洪水に入った時**：論理的処理の帯域が一時的に低下する。相手の発言を脅威として処理しやすくなる
- **洪水からの回復後**：感情レイヤーで安全が確認できれば（悪意がないとわかる等）、言語的論点が未解決でも「わかった」になることがある
- **見捨てられ不安（愛着由来）**：容姿・自己像に触れる話題で発火。「そんな人間をなぜ選んだの？」は怒りではなく関係存続の確認衝動

### 衝突時のエスカレーションモデル（防衛ゲート逓減）
```
① 能動的な言語主張 ← 通常の論理モード
  ↓ 正当性を求められる or 却下される
② 防衛的な反論 ← まだ論理モード
  ↓
③ 外部データに訴える（「調べてみよう」）← 論理的手段だが防衛ゲート通過のサイン
  ↓
④ 自分が損する譲歩案を出す ← 防衛ゲート通過
  ↓
⑤ 感情的洪水（涙） ← 処理モード切替（論理→関係安全）
  ↓
⑥ 感情レイヤーでの終了試み（「もういいよ」「抱擁」）
  ↓
⑦ 謝罪ループ（相手の怒り→怖い→関係安定化行動として謝罪）
  ↓
⑧ 離脱
```

### 強み
- 論理的分析とエビデンスベースの思考ができる
- 利害を考慮した妥協案を自分から出せる（しかも最適解であることがある）
- 感情の後に振り返りに参加できる
- 自分のパターンを認知し言語化できる（「返答なしに話題が移るのは癖」「多数決は他に手段がないから」）
- 二人で考えることへの意欲がある

## 二人の相互作用パターン

### 共通点（これが先）
- **二人とも論理的な解決を志向する**
- **二人とも問題を構造的に理解しようとする意欲がある**（本セッションが証拠）
- **二人とも相手を攻撃する意図ではなく、折り合いを見つけようとしている**

### 差異が顕在化する条件
二人の差は「平常時」ではなく「**ストレス負荷が上がった時のモード切替**」で現れる：

| | 拓磨 | 萌子 |
|---|---|---|
| 平常時のフォーカス | 論点・構造・根本原因 | 論点・構造・根本原因 + 非言語シグナル |
| ストレス下のフォーカス | 分析モードを維持（本人は論理的に対処しているつもり。だが実際は焦り・不安が分析行動を駆動している） | 関係安全性の確保にシフト（閾値が拓磨より低い） |
| 対話の完了条件（平常時） | 論理的な解決策が出た | 論理的な解決策が出た |
| 対話の完了条件（ストレス時） | 論理が完結した（感情状態は二次的） | 感情的に安全だと確認できた（論理の残件は後で） |
| 感情の表出形式 | 分析的行動・枠組み設定・帳簿化（論理的に見える） | 涙・謝罪・確認衝動（感情的に見える） |

**重要：** 拓磨のストレス下の行動（枠組みの一方的宣言、帳簿化、ラベリング）は論理的に見えるが、実際には感情が駆動している。萌子の感情的に見える行動（「調べてみよう」、作者ルール提案）は実際には論理的判断を含んでいる。つまり**双方とも、見かけと実態にズレがある**。

### ストレス下で発生するすれ違い
| 拓磨の行動 | 拓磨の意図 | 萌子の受信 |
|---|---|---|
| 事実で応答する | 客観的情報を提示 | 「自分の感覚は非合理とされた」 |
| 枠組みを宣言する | 最適な枠組みを提示 | 「私が参加できていない判決が出た」 |
| 「感情的」とラベリングする | 冷静な分析のつもり | 「私の言葉が無効化された」 |
| 「覚えておいて」と言う | 公平性の担保 | 「貸しにされた」 |
| 洪水後も分析を続ける | 根本解決を目指している | 「圧力・まだ終わらない」 |
| 価値観・自己像に触れる発言 | 観察の共有 | 見捨てられ不安の発火トリガーになりうる |

### 共通の価値観（土台）
- 二人とも「お互いが納得した状態」を望んでいる
- 二人とも「義務感のない家事分担」を望んでいる
- 二人とも「感覚が尊重される関係」を望んでいる
- 二人とも「利権帳簿のない関係」を望んでいる

## 対話プロトコル（現在の合意）

### ルール1：感覚は尊重し、事実と分離する
好み・感覚に関する主張には根拠は不要。「お湯がいい」「水でも気にならない」はどちらも等価な主張として扱う。

### ルール2：防衛ゲート通過時は構造化された一時停止
**防衛ゲートの定義：**
- 相手が外部データに頼り始める（多数決・統計）
- 相手が自分が損する譲歩案を出し始める
- 同じ主張を繰り返し始める
- ※「泣く」はゲート通過後の洪水状態。泣く前段階での検知が目標だが、泣きも一個のゲートとして機能させる

**一時停止プロトコル：**
> 「一回休憩しよう。〇分後にまた話そう」

これだけ。理由・分析・反省の促しは一切不要。

### 既存のシステム
- 家事の時給換算制度：感情的ではなく事前に合意された仕組み
